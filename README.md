# Bank-Costumer-Churn-Analysis

## Done an  analysis  on bank's costumer dataset based on the raw data provided and a made a dasboard on Power Bi to visualize it.

# What does Churn analysis?
+ It is advantageous for banks to know what leads a client towards the decision to leave the organization/bank.
+ Churn prevention allows companies to develop loyalty programs and retention campaigns to keep as many customers as possible.
